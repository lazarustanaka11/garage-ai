🚗 Garage AI

An AI-powered garage management platform built with FastAPI, PostgreSQL, SQLAlchemy, Bootstrap and OpenAI GPT-5.

Manage customers, vehicles and repair jobs while using AI to generate intelligent repair diagnostics directly from the Repair Jobs module.

👥 Customers 🚘 Vehicles 🤖 AI Repair Jobs Login Register

👥 Customers

Register customers, manage contact information and keep all client records in one place.

🚘 Vehicles

Store vehicle information including make, model, VIN and registration details linked to each customer.

🔧 Repair Jobs

Create repair jobs, track progress and maintain a complete repair history for every vehicle.

🤖 AI Diagnostics

Generate AI-powered repair diagnostics using OpenAI GPT-5. Diagnoses are automatically saved and can be viewed later from the Repair Jobs page.

🚀 Features

  • JWT Authentication
  • Customer Management
  • Vehicle Management
  • Repair Job Tracking
  • OpenAI GPT-5 Diagnostics
  • REST API with FastAPI
  • PostgreSQL Database

💡 About Garage AI

Garage AI is a modern full-stack application demonstrating clean backend architecture, JWT authentication, PostgreSQL persistence and AI integration using the OpenAI Responses API.

The application is designed to help garages manage daily operations while providing intelligent repair assistance through GPT-powered diagnostics.
