# Garage AI

An AI-powered garage management system built with FastAPI.

## Features

- JWT Authentication
- Customer Management
- Vehicle Management
- Repair Job Tracking
- PostgreSQL
- SQLAlchemy 2.0
- Alembic Migrations
- Repository Pattern
- Service Layer Architecture
